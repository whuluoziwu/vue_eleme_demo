# vue_elmshop_demo

## 安装步骤

```
// 把模板下载到本地
git clone https://github.com/whuluoziwu/vue_eleme_demo.git    
// 进入模板目录
cd vue_elmshop_demo
// 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn
npm install         
// 开启服务器，浏览器访问 http://localhost:8080,这一步就可以看到界面了
npm run serve

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build
```
